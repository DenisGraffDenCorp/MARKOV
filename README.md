# MARKOV
Цепи Маркова – это генератор случайных символов, вероятность появления которых зависит от предыдущего символа. Это легко расширяется на общий случай, когда нужно использовать несколько предыдущих символов.  Используйте результат задания 1 и посчитайте вероятности появления:  Буквы после 1, 2, 3, ..., 13 предыдущих букв и в целом.
